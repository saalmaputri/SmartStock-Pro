# SmartStock Pro Backend

FastAPI backend untuk MVP SmartStock Pro.

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
uvicorn app.main:app --reload
```

Swagger tersedia di `http://127.0.0.1:8000/docs`.
