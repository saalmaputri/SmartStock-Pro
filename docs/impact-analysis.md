# Impact Analysis

SmartStock Pro mengganti pencatatan spreadsheet menjadi aplikasi web inventaris. Dampak positifnya adalah data lebih terstruktur, stok dihitung otomatis, laporan lebih cepat, dan aktivitas user tercatat.

Dampak yang perlu dikelola: pelatihan user, pembersihan data awal, dan perubahan kebiasaan kerja dari spreadsheet ke aplikasi.
