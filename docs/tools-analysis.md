# Tools Analysis

- React + Vite dipilih karena cepat untuk demo UI dan mudah dijalankan.
- Tailwind CSS dipilih agar styling konsisten tanpa konfigurasi berat.
- FastAPI dipilih karena dokumentasi API otomatis tersedia di `/docs`.
- SQLite dipilih untuk demo lokal tanpa instalasi server database.
- SQLAlchemy dipilih agar model database siap dimigrasikan.
- Recharts dipilih untuk chart dashboard yang ringan.
- ReportLab dipilih untuk membuat PDF langsung dari backend Python.
- Python `csv` dipilih karena cukup untuk import produk MVP.
