# User Guide

1. Login menggunakan akun demo.
2. Lengkapi master data kategori, supplier, gudang, dan produk.
3. Gunakan menu Inventaris untuk barang masuk.
4. Gunakan menu Inventaris untuk barang keluar.
5. Gunakan tipe Transfer untuk memindahkan stok antar gudang.
6. Cek Dashboard untuk total stok, stok menipis, dan grafik.
7. Gunakan Import CSV di menu Produk untuk menambahkan produk massal.
8. Gunakan Export PDF untuk laporan stok.
9. Gunakan Audit & Logs untuk melihat aktivitas user dan error log.
